"# React-App"

## Node -v

v20.11.1

## Docker postgres

```bash
$ docker compose up
```

## Installation and run backend

```bash
$ cd server
$ npm install
$ npm run start
```

## Installation and run forntend

```bash
$ cd client
$ npm install
$ npm run dev
```

## Run storybook

```bash
$ cd client
$ npm run storybook
```

## Run test front

```bash
$ cd client
$ npm run test
```

## Run test back

```bash
$ cd server
$ npm run test
```
